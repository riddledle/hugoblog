指导：总而言之，caffe是一个深度学习框架，接下来的文章介绍的是它的理念（哲学），结构和用处，使用这个框架之前，最好有机器学习和神经网络的储备知识，但是这儿也会稍微介绍。

哲学：caffe的五个特色：压缩，高速，模块化，开放性，社区化

浏览：目录如下

  • Nets, Layers, and Blobs 网络，层，团: 剖析caffe的模型结构
  • Forward / Backward 前进／后退: 复合模型中必需的计算
  • Loss 损失: 训练计划是用loss定义的
  • Solver 解算机: 实现最优化
  • Layer Catalogue 层目录: 层结构是模型的基本单元
  • Interfaces 界面: 命令行, Python, MATLAB Caffe.
  • Data 数据: 如何调整数据
  • Caffeinated Convolution caffe卷积: 如何计算卷积

深度学习：

这里主要介绍了网上一些可以直接获取的深度学习的资料，特别是和视觉相关的，英文内容，直接阅读。
